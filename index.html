О проекте

FCF Basis — это низкоуровневая библиотека на C++, разработанная для решения задач, требующих гибкости динамических языков при сохранении скорости компилируемого кода. Библиотека предоставляет механизмы для создания универсальных контейнеров, динамической рефлексии типов и сложной системы спецификаторов.

Идеально подходит для создания движков игровых систем, скриптовых интерпретаторов и высоконагруженных серверных приложений.

Ключевые возможности

⚡

Variant & Dynamic Types

Класс BasicVariant с поддержкой Small Buffer Optimization (SBO). Хранит данные любого типа без лишних аллокаций для малых объектов.

🧩

Container Access

Унифицированный интерфейс итерации для любых контейнеров (std::vector, std::map, std::list) через ContainerAccess и Cursor.

🔍

Runtime Reflection

Полная информация о типах в рантайме через TypeInfo. Поддержка конвертации типов и динамического создания объектов.

🛠️

Specificators

Расширяемая система спецификаторов для обработки данных (например, MinMaxSpecificator для генерации случайных чисел).

Техническое описание

1. Архитектура Variant

Ядром библиотеки является шаблонный класс BasicVariant<InnerBufferSize>. Он использует технику Small Buffer Optimization для хранения небольших типов (int, float, short strings) непосредственно внутри объекта, избегая выделения памяти в куче.

template <size_t innerBufferSize>
class BasicVariant {
    public:
        enum DataSetMode { WRITE, RESET, REFERENCE, FORCE_REFERENCE };

        template <typename Ty>
        BasicVariant(const Ty& a_value);

        template <typename Ty>
        Ty cast() const;

    private:
        const TypeInfo* _typeInfo;
        void*           _ptr;
        char            _mem[innerBufferSize]; // SBO Buffer
};

2. Система типов (Type System)

Каждый тип регистрируется в глобальном хранилище TypeStorage. Структура TypeInfo содержит метаданные, включая имя типа, размер, указатели на функции конвертации и спецификаторы.

  • Динамическая регистрация: Макросы и шаблоны автоматически регистрируют типы при первом использовании.
  • Конвертация: Поддержка конвертеров между типами через ConvertFunction.
  • Идентификация: Уникальные индексы типов с битовыми флагами для константности и ссылок.

3. Итерация и Контейнеры

Библиотека абстрагирует доступ к элементам контейнеров через класс ContainerAccess. Это позволяет писать алгоритмы, работающие с std::vector, std::map или пользовательскими структурами одинаково.

template <typename TContainer>
class ContainerAccess {
    public:
        typedef typename CursorSelector<const TContainer>::Type cursor_type;

        inline void setBeginPosition();
        inline void setEndPosition();
        inline void inc();
        inline void dec();

        inline bool isEnd() const;
        inline void setValue(const Variant& a_value);
};

4. Функциональность

Включает набор утилитарных функций, таких как fill и random, которые могут работать как с обычными указателями, так и с динамическими контейнерами.

  • random(Ty1 min, Ty2 max) — генерация случайных чисел.
  • fill(begin, end, value) — заполнение диапазона.

Пример использования

Создание динамического контейнера и итерация по нему:

// Создание варианта с типом std::vector<int>
Variant vec = Variant(std::vector<int>{1, 2, 3, 4, 5});

// Получение итератора
Variant::iterator it = vec.begin();

while (it != vec.end()) {
    // Доступ к значению
    int val = it.value().cast<int>();

    // Изменение значения
    it.setValue(val * 2);

    ++it;
}
