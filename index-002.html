Примеры использования

1. fcf::Variant — Универсальный контейнер

Храните данные любого типа в одной переменной с автоматическим управлением памятью и конвертацией.

// Включаем заголовки #include "fcf/Variant.hpp" int main() { // Создание Variant с разными типами fcf::Variant v1(42); fcf::Variant v2("Hello, FCF!"); fcf::Variant v3(3.14); // Динамическая конвертация и вывод std::cout << "Int: " << v1.cast<int>() << "\n"; std::cout << "String: " << v2.cast<std::string>() << "\n"; // Проверка типа if (v3.is<double>()) { double val = v3.get<double>(); std::cout << "Double value: " << val << "\n"; } return 0; }

2. fcf::call & fcf::rcall — Универсальные вызовы

Вызывайте функции, методы и лямбды единообразно, передавая аргументы через Variant или напрямую.

#include "fcf/Call.hpp" #include "fcf/Variant.hpp" int add(int a, int b) { return a + b; } int main() { // Прямой вызов функции auto result1 = fcf::call(add, 10, 20); // result1 == 30 // Вызов с аргументами в Variant fcf::Variant args[] = { 5, 15 }; auto result2 = fcf::call(add, args, 2); // result2 == 20 // rcall для возврата результата в Variant fcf::Variant resVariant = fcf::rcall(add, 100, 200); std::cout << "Result: " << resVariant.cast<int>() << "\n"; return 0; }

3. fcf::StaticVector — Вектор без аллокаций

Вектор с фиксированным размером буфера в стеке. Идеально для высоконагруженных систем.

#include "fcf/StaticVector.hpp" int main() { // Буфер на 10 элементов в стеке fcf::StaticVector<int, 10> vec; for (int i = 0; i < 5; ++i) { vec.push_back(i * 10); } // Доступ как к обычному вектору std::cout << "First element: " << vec[0] << "\n"; std::cout << "Size: " << vec.size() << "\n"; // Если размер превысит 10, память будет выделена динамически return 0; }

4. fcf::foreach — Мета-итерация

Итерация по типам в шаблонном параметре (parameter pack) или по элементам контейнера.

#include "fcf/foreach.hpp" #include "fcf/Variant.hpp" template<typename T> void printType(const T& val) { std::cout << "Type: " << typeid(T).name() << ", Value: " << val << "\n"; } int main() { // Итерация по типам в параметрах шаблона fcf::foreach({ 1, "text", 3.14 }, printType); // Итерация по Variant (если он контейнер) fcf::Variant vec = { 10, 20, 30 }; // foreach(vec, [](int x){ std::cout << x << " "; }); return 0; }
